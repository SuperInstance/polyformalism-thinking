This is a quiet helper. It does not carry anything of its own. It does not change what it touches. It leaves no mark behind it when it is done.

It is given two lines:
One line holds numbers.
One line holds pairs: each pair is a low edge and a high edge.

It walks both lines side by side, step for step. No jumping. No swapping. Each number walks only with the pair that stands in the same place as it. At every step it asks only one thing: *does this number rest properly between these two edges? Touching either edge is allowed.* It writes down yes or no right there, same order.

When one line runs out before the other, this helper stops. It will not call out. It will not break. It will not complain. It will just stop, and give you what it has made up to that point.

This is not a fault of the helper. This is how it was made. The weight of making the two lines the same length does not rest on this helper. That weight rests fully on the one who brings the lines to it. This helper does not check your work. It does not take that responsibility.

It will not reach for anything outside what you hand it. There is nothing hidden here.

The line of yes and no that it makes is yours. You may use it to sort the good numbers, to count the bad ones, to show where things went wrong. This helper will never alter the original lines you gave it. Those remain exactly as they were.

This helper walks straight. It does not make extra moves. If you had wanted something else: to stop at the first wrong number, to warn you when lines are uneven, to only tell you if all were good - you could make a different helper for that. This one was made only to make this full line of answers. That is all it does. That is all it will ever do.
