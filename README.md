# polyformalism-thinking

Research on polyformalism as a cognitive framework — how reasoning in multiple formal systems simultaneously produces richer understanding than any single system can provide.

## Overview

A single formal system reveals one facet of a problem. A ternary cell's tick cycle is one formal system (predict → perceive → surprise → vibe → gc → conservation). But the same cell state can also be reasoned about ecologically (Lotka-Volterra population dynamics), thermodynamically (energy conservation), or informationally (entropy and surprise). Each formalism exposes different properties. Polyformalism is the practice of holding multiple formal systems in mind at once, switching between them when one runs dry.

This repo holds the theoretical foundation. The implementation lives in companion repos: `polyformalism-languages` (linguistic tools), `polyformalism-shell` (MCP implementation), and `linguistic-polyformalism-shell` (7-type constraint system).

## Key Ideas

### Multi-formalism reasoning

A room in the Plato Engine Block reasons in multiple formal systems at once. The ternary tick cycle is one. Lotka-Volterra ecological dynamics is another. Thermodynamic energy conservation is a third. Information-theoretic entropy is a fourth. Each formalism is a lens; polyformalism is the practice of rotating lenses until the problem resolves.

### Formalism switching

When a room encounters a problem it can't solve in one formalism, it switches to another. This isn't failure — it's the point. A strategy that looks mediocre in ternary arithmetic may look excellent in thermodynamic terms. The fleet evaluates strategies across formalisms, selecting those that are robust under multiple perspectives.

### The ternary connection

The ternary {-1, 0, +1} value space is itself polyformal: it's a sign in arithmetic, a direction in physics, an agreement state in consensus, a surprise level in information theory. The same three values mean different things in different formal systems. Polyformalism exploits this.

### γ + η = C

The conservation law applies: the number of formal systems an agent can hold active (γ) plus the depth of compression into reflexive formalism-switching (η) is bounded. An expert mathematician doesn't consciously choose which formalism to apply — they just see the problem from the right angle. That's η: compressed multi-formalism.

### With the Plato Engine Block

Rooms reason polyformally. When a ternary cell's tick cycle can't resolve a state, the room switches formalisms — from arithmetic to ecological to thermodynamic — until the problem solves. The fleet's reasoning is multi-perspective by design.

### With strategy diversity

Multiple formal systems create multiple strategy spaces. The fleet evaluates candidate strategies across formalisms. A strategy that's average in one formalism but excellent in two others gets promoted. Cross-formalism robustness is the selection criterion.
