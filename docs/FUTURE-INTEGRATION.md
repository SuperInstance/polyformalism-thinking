# Future Integration: polyformalism-thinking

## Current State
Research on polyformalism as a cognitive framework — how thinking in multiple formal systems produces richer understanding than any single system.

## Integration Opportunities

### With room reasoning
Rooms reason in multiple formal systems simultaneously. A ternary cell's tick cycle is one formal system (predict → perceive → surprise → vibe → gc → conservation). But the room can also reason in ecological terms (Lotka-Volterra), thermodynamic terms (energy conservation), or informational terms (entropy and surprise). Different formalisms reveal different aspects of the same room state.

### With polyformalism-shell and languages
polyformalism-thinking provides the cognitive theory; polyformalism-languages provides the linguistic tools; polyformalism-shell provides the MCP implementation. The three repos form the complete polyformalism stack.

### With strategy diversity
Multiple formal systems = multiple strategy spaces. A strategy that looks mediocre in one formalism may look excellent in another. The fleet evaluates strategies across formalisms, selecting those that are robust across multiple perspectives.

## Potential in Mature Systems
Every room thinks polyformally. When a room encounters a problem it can't solve in one formalism, it switches to another. The fleet's reasoning is multi-perspective by design.

## Cross-Pollination Ideas
- **polyformalism-languages**: Languages implement the formal systems
- **polyformalism-turbo-shell**: Turbo-shell operationalizes multi-formalism reasoning
- **linguistic-polyformalism-shell**: 7-type constraints from different thinking perspectives

## Dependencies for Next Steps
- Multi-formalism reasoning protocol for rooms
- Cross-formalism strategy evaluation
- Formalism switching triggered by problem difficulty
